# Gatsby Plugin with Material UI

### 1. Install

```bash
npm install @material-ui/core
npm install gatsby-plugin-material-ui
```

### 2. Edit gatsby-config.js

```bash
module.exports = {
  plugins: [`gatsby-plugin-material-ui`],
};
```

### 3. Use it!

```jsx
<Card className={classes.root}>
    <CardActionArea>
	      <img
	        alt="Greate Supporter"
	        className={classes.media}
	        src={profile}
	        title="Greate Supporter"
		    />
	      <CardContent>
		        <Typography gutterBottom variant="h5" component="h2">
			          Someone who want to go with me!
		        </Typography>
		        <Typography variant="body2" color="textSecondary" component="p">
			          But there is no replyed.
		        </Typography>
	      </CardContent>
    </CardActionArea>
</Card>
```

Now we can use it just like React!
